9.2 A second attempt at analyzing 0,1 responses: the generalized linear model

Consider the above model again: the linear model has a systematic part, \(\alpha + \beta \times c\_set\_size\), and a random part, the residual \(\varepsilon\), which is assumed to be normally distributed with standard deviation \(\sigma\).

Suppose that the systematic part is written as \(\eta\) (pronounced eta):

\[\begin{equation} \eta = \alpha + \beta \times c\_set\_size \end{equation}\]

As shown in the model fit above, \(\eta\) gives estimate of the proportion of correct responses \(\pi\) for different set sizes:

  • When set size is zero (the average set size), this estimated proportion is \(\hat \alpha = 0.87\)
  • When set size increases by one, this estimated proportion is \(\hat \alpha + \hat \beta = 0.81\).

The link between \(\pi\) and \(\eta\) is trivial: they are identical.

\[\begin{equation} \pi = \eta \end{equation}\]

The above identity relation states that \(\pi\) is a function of \(\eta\), so we can write this as

\[\begin{equation} \pi = g(\eta) \end{equation}\]

In the generalized linear modeling framework, the function \(g(\cdot)\) is called the link function. In the familiar linear model, it is the trivial identity function: \(\pi = \eta\) or \(\pi = g(\eta)\).

Now, the link function \(g(\cdot)\) need not be the identity function. In our example analysis here, the goal is to estimate (i) the proportion \(\pi\) of correct responses when the set size is zero (the average set size), and (ii) the change in the proportion of correct responses with a unit increase in set size. In order to estimate the two proportions, we define the following link function:

\[\begin{equation} \eta = g(\pi) = \log(\frac{\pi}{1-\pi}) \end{equation}\]

This link function is called the logit link function. The term \(\frac{\pi}{1-\pi}\) is called the odds.1 The logit link function is a log-odds; it real numbers ranging from \((-\infty,+\infty)\) to probability values, which range from \([0,1]\); see figure 9.1.

FIGURE 9.1: Logistic function

Having defined a link function, we fit the model on the log-odds scale:

\[\begin{equation} \eta = \log(\frac{\pi}{1-\pi}) = \alpha + \beta \times c\_set\_size \end{equation}\]

Once \(\eta\) is estimated, one can easily compute the estimated probabilities of interest by solving the above equation for \(\pi\) (in other words, by computing the inverse of the function \(g(\cdot)\):

\[\begin{equation} \pi = g^{-1} \log(\frac{\exp(\eta)}{1+\exp(\pi)}) = \alpha + \beta \times c\_set\_size \end{equation}\]

Returning to our research question, the estimates for \(\eta\) are computed as follows. In the code below, we define the link function by writing family=binomial().

m_logit<-glm(correct~c_set_size,
    df_recall_data,family=binomial())
summary(m_logit)

The estimate \(\hat\eta\), in log-odds units, for the average set size is 2.89; back-transforming this to proportions, we have \(\frac{\exp(\hat\alpha)}{1+\exp(\hat\alpha)}=0.94735\).

Similarly, the proportion corresponding to one unit increase in set size is \(\frac{\exp(\hat\alpha+\hat\beta)}{1+\exp(\hat\alpha)}=0.94735\).

  1. Odds are always a ratio of the probability of success to the probability of failure. For example, the odds of obtaining a one in a fair six-sided die are =1/5. The odds of obtaining a heads in a fair coin are 1/1.↩
